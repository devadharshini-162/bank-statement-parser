What you have built — done and working
PDF Intelligence Layer PyMuPDF detects digital vs scanned PDFs automatically. Tesseract OCR as fallback. Gemini Vision for scanned pages. Works on any Indian bank format.
AI Extraction Engine Gemini 2.5 Flash reads raw text or images and returns structured JSON. Handles multi-date rows, UPI descriptions, ATM entries, interest credits — all edge cases.
FastAPI Backend REST API with /parse and /download endpoints. File validation, UUID-based temp storage, anomaly detection, Excel export with auto-formatting. Swagger docs auto-generated.
React Frontend Dark fintech UI with drag-and-drop upload, animated loading states, stat cards, transaction table with search, download button. Lucide icons, Cabinet Grotesk font.
Data Processing Pandas cleans and rounds financial values to 2 decimal places. Validates totals, flags anomalies — transactions with neither debit nor credit, negative balances.
GitHub Full project live at github.com/devadharshini-162/bank-statement-parser with proper commits, .gitignore, README. Clean commit history from day one.
